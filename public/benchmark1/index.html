Farm Defense Game Document

INTRODUCTION

This document describes a game called “Farm Defense”, which is designed to be a simple 2D top-down game. The game will use sprite-based animation, characters, tiled, backgrounds, collision detection, AI, search algorithm and showing memory management.

TECHNOLOGY

For our technology, the game “Farm Defense” will be developed for the Windows Platform using the Wofile 2D engine, which is the basic game engine made by Stony Brook University.

BACK STORY

One day, strange occurrences began happening around a peaceful village, accompanied by unusual sounds emanating from the forest behind it. New, unidentified types of insects emerged, ravaging crops and decimating farms without warning. These insects seemed to possess a mysterious energy, becoming even larger and more menacing under the moonlight. At this time, Kevin decided to transform the harvested crops into defense towers to protect the village. Will Kevin be able to hold the fate of the village and safeguard his farm from these unknown insects?

OBJECTIVE

Collect seeds, harvest and Defense farm from the bug monsters

GAMEPLAY

- The game operates like a top-down view, where the protagonist can move left, right, up, and down around the platform, and interact with specific tiles.

- At the start : You collect seeds necessary for defense and check monster information, cultivate the land to harvest the desired crops, and then install these crops for defense.

- After the timer ends: You battle an oncoming horde of insects using the crops you've installed. If necessary, you can install additional crops or relocate the ones already placed.

CONTROLS

Game played using Keyboard.

A - Move Left
D - Move Right
W - Move Up
S - Move Down
Q - Pick Up Item / Pickup plants
E - Use Item / Install plants
ESC - If the game is in process, pause the game and show [ continue, quit, restart ].
